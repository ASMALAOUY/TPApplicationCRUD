# TP Application CRUD - Express.js, MongoDB & Mongoose

## Description

Ce projet est une application web complète qui implémente les opérations CRUD (Create, Read, Update, Delete) pour la gestion de produits. L'application permet d'ajouter, modifier, supprimer et visualiser des produits avec des fonctionnalités avancées comme la recherche, les filtres et la pagination.

## 🛠 Technologies Utilisées

- **Node.js** - Environnement d'exécution JavaScript
- **Express.js** - Framework web pour Node.js
- **MongoDB** - Base de données NoSQL
- **Mongoose** - ODM pour MongoDB
- **EJS** - Moteur de templates
- **Bootstrap 5** - Framework CSS pour l'interface
- **Docker** - Conteneurisation de MongoDB

##  Fonctionnalités

-  Créer un nouveau produit
-  Lire/consulter tous les produits
-  Afficher les détails d'un produit
-  Modifier un produit existant
-  Supprimer un produit
-  Recherche de produits
-  Filtrage par catégorie
-  Filtrage par prix (min/max)
-  Pagination des résultats
-  Gestion des tags
-  Messages flash pour les actions
